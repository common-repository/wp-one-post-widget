== Description ==

This plugin show only one post in the sidebar or widgetzed area.

== Installation ==

1. Upload `wp-one-post-widget` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the `Plugins` menu in WordPress.
3. In editing the post, options the screen, enable the post excerpt.
4. Write the excerpt to appear the text with link read more.
5. If you prefer, add the image highlighted to display in the post.
6. In widgets menu, drag for the sidebar and search the post to display on your website.
